# Commands Guide

SuperDroid provides 30 slash commands organized by category.

## Planning & Design

### /brainstorm
Structured brainstorming with Socratic questioning.
```
/brainstorm "new feature idea"
/brainstorm improving user experience
```

### /design
System architecture and design.
```
/design user authentication system
/design --type api payment processing
```

### /estimate
Time and effort estimation.
```
/estimate implementing OAuth2
/estimate database migration
```

### /spec-panel
Multi-expert specification review.
```
/spec-panel "user registration requirements"
```

## Development

### /implement
Feature and code implementation.
```
/implement user login --type component
/implement REST API --with-tests
```

### /build
Build and compilation workflows.
```
/build
/build --env production --clean
```

### /improve
Code improvements and optimizations.
```
/improve src/utils/helpers.js --focus performance
```

### /cleanup
Code refactoring and cleanup.
```
/cleanup src/ --scope imports
/cleanup . --scope all
```

### /explain
Code explanation and documentation.
```
/explain src/auth/jwt.ts
/explain this regex pattern
```

## Testing & Quality

### /test
Test generation and execution.
```
/test user authentication --type unit
/test checkout flow --type e2e --coverage
```

### /analyze
Code analysis for quality and patterns.
```
/analyze src/ --focus security
/analyze components/ --focus performance
```

### /troubleshoot
Debugging and issue resolution.
```
/troubleshoot "TypeError: Cannot read property..."
/troubleshoot build failing on CI
```

### /reflect
Retrospectives and learning.
```
/reflect on sprint 5
/reflect implementing authentication
```

## Documentation

### /document
Generate documentation.
```
/document src/api/ --type api
/document . --type readme
```

### /help
Show SuperDroid help.
```
/help
/help research
```

## Version Control

### /git
Git operations.
```
/git status
/git commit "feat: add authentication"
```

## Project Management

### /pm
Project manager mode with PDCA.
```
/pm build user management
/pm --strategy brainstorm
```

### /task
Task tracking with TodoWrite.
```
/task add "implement login"
/task list
/task complete 1
```

### /workflow
Workflow automation.
```
/workflow dev
/workflow release --version 1.0.0
```

## Research & Analysis

### /research
Deep web research.
```
/research "GraphQL best practices" --depth deep
/research "competitor analysis"
```

### /business-panel
Multi-expert business analysis.
```
/business-panel "market expansion strategy"
```

## Utilities

### /agent
Invoke specialized droids.
```
/agent security-engineer "audit auth code"
```

### /index-repo
Repository structure analysis.
```
/index-repo --depth deep
```

### /index
Quick file/directory overview.
```
/index src/
```

### /recommend
Get command recommendations.
```
/recommend "I need to add authentication"
```

### /select-tool
Tool selection guidance.
```
/select-tool "find deprecated API usage"
```

### /spawn
Parallel task execution.
```
/spawn ["analyze frontend", "analyze backend"]
```

### /load
Load saved session.
```
/load auth-feature
```

### /save
Save current session.
```
/save checkpoint-1
```

### /sd
Quick reference for all commands.
```
/sd
```
